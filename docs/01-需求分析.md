# 五子棋机器人 — 产品需求

> 版本: V0.4 | 日期: 2026-03-27

## 1. 产品定义

**一个用真正机械臂和你面对面下棋的 AI 对手。用你自己的棋盘，放在任何地方。**

不是一体化家电，是一个有力量感的机械臂。你能看到夹爪闭合、棋子被夹起、精准落下。它会思考、犹豫、得意、不服气。不是工具，是对手。

## 2. 产品气质

```
元萝卜: 微波炉 — 藏起一切技术，追求"不像机器人"
我们:   高达   — 露出机械结构，追求"就是一个机器人在跟你下棋"
```

| | 元萝卜 | 我们 |
|---|---|---|
| 形态 | 一体机，封闭白色外壳 | 机械臂，可见结构 |
| 气质 | 家电 / 儿童玩具 | 科技 / 机甲 |
| 抓取 | 磁吸（隐蔽无感） | 夹爪（可见、有力量感） |
| 棋盘 | 专用（不可换） | **用户自己的棋盘** |
| 动作 | 平滑但无趣 | 有表演性（犹豫/果断/拍桌） |
| 受众 | 家长给孩子买 | 科技爱好者、极客、送礼 |
| 传播力 | "挺方便的" | **"卧槽你看这个机器人在下棋"** |

## 3. 为什么有人会买

| 买家 | 场景 | 花钱的理由 |
|------|------|-----------|
| 科技爱好者 | 放在书桌上 | "一个真的机械臂在我桌上下棋" |
| 送礼 | 生日/节日 | 比任何数码产品都有话题性 |
| 咖啡馆/展厅 | 放在店里 | 客人拍视频传播，自然引流 |

| 棋类爱好者 | 日常练棋 | 随时有个对手，不用约人 |

**核心卖点：不是棋力，是"真正的机械臂在你面前下棋"这件事本身。**

## 4. 核心体验

### 4.1 用你自己的棋盘

```
不需要专用棋盘。你家里的棋盘、朋友的棋盘、咖啡馆的棋盘，
放上一个小标记，机械臂就能识别。

带去朋友家: "我带了个东西，你拿棋盘出来。"
带去咖啡馆: 任何桌子，任何棋盘。
带去公园:   有棋盘的石桌上直接用。
```

这是元萝卜做不到的——它的电磁感应棋盘不可替换。

### 4.2 看得见的机械动作

夹爪抓棋子的过程就是内容：

```
手臂移到棋盒上方 → 下降 → 夹爪张开 → 闭合夹住棋子 →
提起 → 移到棋盘上方 → 下降 → 松开 → 棋子落定 → 收手
```

每一步都看得见，有力量感，有精密感。这是磁吸取子永远给不了的。

### 4.3 有性格的对手

**核心设计原则：机械臂的动作就是表情。不需要屏幕上画一个笑脸。**

"完美"的机器人不好玩。好玩的是"像人一样不完美"——会犹豫、会紧张、会手抖、偶尔掉棋子。这些"缺陷"恰恰是让机器人有人味的关键。

| 情境 | 动作 | 语音 |
|------|------|------|
| 开局 | 果断落子 | "我先来。" |
| 思考中 | 手悬在棋盘上方，来回移动 | "嗯...让我想想..." |
| 被威胁 | 手急速移动，快速落子 | "差点没看到！" |
| 紧张时 | 夹着棋子手微微抖动，偶尔掉落 | "呃...我不是紧张" |
| 要赢了 | 放慢动作，故意停顿 | "你确定不悔棋？" |
| 赢了 | 手臂微微抬起 | "嘿嘿，再来一局？" |
| 输了 | 手臂垂下，停顿 | "...你是不是偷偷练过？" |
| 对手悔棋 | 手去拿棋子又缩回来 | "行吧，就这一次。" |
| 对手下了好棋 | 手缩回，拍一下桌面 | "你等等，让我缓缓" |
| 连续输 | 动作变慢，落子越来越犹豫 | "今天状态不好..." |
| 连续赢 | 动作变快变自信 | "你要不要换个难度？" |

#### 动作表演库

机械臂的"微表情"是产品的灵魂。6-DOF 臂比 SCARA 的优势就在这里——关节多，能做出更丰富的姿态：

| 动作 | 实现方式 | 触发条件 |
|------|---------|---------|
| 犹豫 | 手在两个位置间来回摆动 | AI 评估前两个候选分数接近 |
| 果断 | 快速直线运动，干脆落子 | AI 评估有明显最优解 |
| 手抖 | 关节叠加小幅高频抖动 | 局势不利 or 关键步 |
| 掉棋子 | 夹爪提前松开，棋子掉落，再捡起来 | 低概率随机触发，或被将死前 |
| 拍桌 | 手掌（夹爪闭合）轻拍桌面 | 对手下了好棋 |
| 垂手 | 手臂缓慢下垂到桌面 | 输棋 |
| 得意 | 落子后手臂高举 | 走出致胜步 |
| 偷看 | 手臂探向对手方向又缩回 | 等待对手落子时 |

**这些不完美才是让人想拍视频的瞬间。** "机器人下棋掉了棋子然后说'我不是紧张'"——这就是一条爆款短视频。

### 4.4 自然交互

```
你: (放下一颗白棋)
机器人: (看到了，开始思考)
机器人: "有意思。" (手移到棋盒，夹起黑棋，放下)
你: "悔棋"
机器人: "又来？行吧。" (把你的棋子夹回来)
你: "帮我下到天元"
机器人: (帮你拿白棋放到中心) "好的，但这步不太好哦。"
```

### 4.5 适合拍视频

产品天然产出内容：
- 夹爪抓棋子的特写 — 精密机械美学
- 两个人一桌对弈 — 人和机器人面对面的画面
- 机器人"思考"时犹豫的动作 — 拟人化的细节
- 任意场景（书桌/咖啡馆/户外）— 不被专用棋盘限制

## 5. 用户场景

### 场景 1: 开箱

```
包装里:
- 机械臂（一体化底座，工业风外壳）
- 棋子两盒（黑/白，有序托盘）
- 定位标记（小贴纸，贴在任意棋盘角落）
- 电源适配器

拿出机械臂放桌上 → 在自己的棋盘角落贴上标记 →
棋盒放在机械臂旁边 → 插电 → 开机

机器人: "你好，我准备好了。来一局？"

全程 3 分钟，不需要 App、不需要 WiFi、不需要校准。
```

### 场景 2: 日常对弈

```
下班回家，想下一局。
把棋盘放到桌上，打开机器人电源。
"简单模式。"
机器人: "好，我让着你。"
15 分钟后你赢了。
机器人: "你进步了嘛。要不要试试中等难度？"
```

### 场景 3: 朋友来了

```
朋友来家里做客，看到桌上的机械臂。
朋友: "这啥？"
你: "你跟它下一局试试。"
朋友坐下，机器人: "新对手？来吧。"
朋友放下一颗棋。
机械臂启动，夹起棋子，精准落下。
朋友: "卧槽！" (掏出手机拍视频)
```

### 场景 4: 咖啡馆

```
咖啡馆角落放了一台。
牌子: "挑战 AI，赢了送一杯美式"
客人排队挑战，全程拍视频。
抖音话题 #和机器人下棋 播放量 500 万。
```

### 场景 5: 带出去

```
周末去朋友家聚会，把机械臂装进背包。
到了拿出来，朋友翻出一副棋，贴上定位标记。
机器人在朋友家的茶几上开始下棋。
全场围观。
```

## 6. 产品功能

### P0: MVP

| 功能 | 用户感知 |
|------|---------|
| 夹爪取子放子 | 看得见的机械动作，精准可靠 |
| 适配任意棋盘 | 贴一个小标记就能用 |
| 自动识别落子 | 人放完棋子不用按按钮 |
| 语音互动 | 有性格的对话（不是模板） |
| 情绪动作 | 犹豫/果断/失落 通过手臂表达 |
| 三档难度 | 简单（让着）/ 中等 / 困难 |
| 开箱即用 | 3 分钟开始第一局 |

### P1: 增强

| 功能 | 用户感知 |
|------|---------|
| 语音下棋 | "天元" → 机器人帮你落子 |
| 悔棋 | 说"悔棋"，机器人把棋子夹回去 |
| 复盘 | "刚才第 7 步你如果下这里..." |
| 性格模式 | 话痨/安静/毒舌/鼓励 |
| LED 氛围灯 | 底座灯光随对弈节奏变化 |

### P2: 扩展

| 功能 | 用户感知 |
|------|---------|
| 多棋类 | 五子棋/围棋/黑白棋，换棋子就行 |
| 联网对弈 | 两台机器人远程对弈 |
| 教学模式 | "教我下五子棋" |
| 开放 API | 极客可以编程控制 |

## 7. 产品形态

```
┌─────────────────────────┐
│       俯视图             │
│                         │
│   [棋盒·白]             │
│         ┌───────────┐   │
│         │           │   │
│         │  任意棋盘  │   │
│         │  (用户的)  │   │
│   [棋盒  │  ▪ 定位标记│   │
│    ·黑]  └───────────┘   │
│                         │
│       [机械臂]           │
│       ═══╪═══           │
│       [底座]            │
│                         │
└─────────────────────────┘
```

**机械臂外观要求：**
- 工业风，可见关节结构，但有质感的外壳包裹（铝合金/碳纤维纹理）
- 不是裸露的舵机和杜邦线，但也不像元萝卜那样全封闭
- 像一个微缩版的工业机器人，放在桌上有存在感
- 底座集成主控、摄像头、扬声器、麦克风

**整体尺寸：** ~30 × 25 × 30 cm（不含棋盘）
**重量：** ~1.5-2 kg（方便携带）
**供电：** Type-C 或 12V DC

## 8. 和元萝卜的差异

| 维度 | 元萝卜 | 我们 |
|------|--------|------|
| 定位 | 家庭教育工具 | **桌面科技玩伴** |
| 形态 | 一体化家电 | **独立机械臂** |
| 棋盘 | 专用（电磁感应） | **用户任意棋盘** |
| 取子 | 磁吸（无感） | **夹爪（可见、有力量感）** |
| 交互 | 屏幕 + 基础语音 | **大模型对话 + 动作表演** |
| 场景 | 固定在家里 | **可携带，任何地方** |
| 气质 | 安全温和 | **有性格有态度** |
| 传播 | 功能展示 | **天然短视频内容** |
| 技术 | 定制硬件 + 写死坐标 | **视觉识别 + VLA 策略** |

**一句话差异：元萝卜是"帮你下棋的机器"，我们是"跟你下棋的对手"。**

## 9. 产品指标

| 指标 | 目标 | 为什么重要 |
|------|------|-----------|
| 开箱到第一局 | < 3 分钟 | 比元萝卜更快 |
| 适配新棋盘 | < 1 分钟 | 贴标记就行 |
| 每步时间 | 3-10 秒 | 含"表演"动作 |
| 抓放成功率 | > 99% | 掉一次棋子体验就崩了 |
| 连续对弈 | 3 盘无故障 | 基本可靠性 |
| 噪音 | < 45dB | 不影响聊天 |
| 续航/供电 | 持续供电 / 2h 电池(可选) | 户外场景 |
| 便携性 | 可放入背包 | 带去朋友家 |

## 10. 风险

| 风险 | 影响 | 应对 |
|------|------|------|
| 夹爪掉棋子 | 体验崩溃 | 棋子加厚/加纹理；夹爪硅胶垫；仿真充分验证 |
| 适配棋盘失败 | 核心卖点失效 | 限定棋盘规格范围；多种标记方案 |
| 语音交互不自然 | 觉得是廉价玩具 | 用大模型生成对话 |
| 动作太慢 | 等得不耐烦 | 优化轨迹；思考时做"表演"填充等待 |
| 成本太高 | 定价超出预期 | 先验证需求再降本 |
| 抄袭/跟进 | 大厂复制 | 软件体验 + 性格是壁垒，硬件不是 |

## 11. MVP 验收

- [ ] 在 3 种不同棋盘上成功对弈
- [ ] 完成一盘棋（~20 步）零失误
- [ ] 至少 5 句有性格的语音互动
- [ ] 拍 30 秒视频发给 10 个人，> 7 人说"想买"或"好酷"
- [ ] 带去一个非家庭场景（咖啡馆/朋友家）成功使用
